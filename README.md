# My Movie Project

A website dedicated to cinematography, created as a practical project during front-end development studies. The site presents a collection of favorite movies and series, including detailed information about the production "Taxi Driver".

## Demo
You can view the live project here: https://zuletaxi.github.io/my-project/

## Technologies Used
* HTML5 – content structure and page semantics.

## Project Structure
* index.html – main landing page.
* /pages – subpages with descriptions of specific titles (e.g., About, Movie list).
* /images – graphic resources, movie posters, and icons.

## Site Features
- Responsive navigation menu.
- Clear section layouts with movie descriptions.
- Links to external movie resources.
- Image gallery related to the theme of cinema.
